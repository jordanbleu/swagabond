# HrefV1

> *A hyperlink, containing text and a URL.* 


### `IsEmpty`

If this is true, the link is empty.



* ⚒️ Underlying Type: Boolean



### `Text`

The text for the link



* ⚒️ Underlying Type: String



### `Url`

The URL for the link



* ⚒️ Underlying Type: String
